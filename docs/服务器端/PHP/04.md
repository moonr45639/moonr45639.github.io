## 查看数据

<br>

使用 **var_dump** 方法可以查看变量的详细情况 ：数据大小（ 长度 ），数据类型

<br>

~~~php
$a = 100;
$b = 'hello';
var_dump($a, $b);	//	可以多个一起

// 打印结果
int(100)	// int，值 100
string(5) "hello"	//	string，长度 5，值 hello
~~~

## 基本数据类型

<br>

### 整型 ：int

<br>

**PHP** 数值是有符号的（ 支持正负数 ）

<br>

十进制 ：默认

<br>

八进制 ： 以 **0** 开头

<br>

十六进制 ： 以 **0x** 开头

<br>

二进制 ： 以 **0b** 开头

<br>

- 保持原有的进制输出

    <br>

    ​虽然定义数据的时候可以使用多种进制表示，但是显示数据的时候 **PHP** 默认使用十进制输出

    <br>

    ​想要保持原有的进制输出就要使用 **print** 

    <br>

    ​十进制 ：使用 **%d**

    <br>

    ​八进制 ：使用 **%o**

    <br>

	十六进制 ：使用 **%x**

    <br>

    ​二进制 ：使用 **%b**

<br>

~~~php
print('5的二进制是：%b', $a);		// 输出结果101
~~~

<br>

- 进制转换

    <br>

    ​**PHP** 提供了一套以十进制为基准的转换函数，利用十进制与其他进制组合成单词即可实现函数转换

<br>

### 浮点型 ：float

<br>

有小数点的数、科学计数法或数值超过整型所能表达的最大数

<br>

浮点数适合运用在对精度要求不高的地方

<br>

浮点数不适合进行运算后的精确比较

<br>

### 布尔型 ：boolean

<br>

布尔值不区分大小写

<br>

通常不用来储存数据，而是程序用来进行过程或者结果的判定

<br>

### 字符串 ：string

<br>

PHP 中字符串有多种定义方式

<br>

- 单引号

    <br>

    使用单引号包裹字符串

    <br>

    最简单的字符串，可以解析少量转义字符

<br>

~~~php
$str = 'fun';
~~~

<br>

- 双引号

    <br>

    使用双引号包裹字符串

    <br>

    双引号相对于单引号功能较多，可以解析较多的转义字符，其中包含变量时也会被解析

    <br>

    变量名尽量不要跟其他字母或数字保留在一起，可以在变量与后面內容之间加一个空格或者将变量用大括号包裹起来

<br>

~~~php
$str = "fun";
~~~

<br>

- 定界符

    <br>

    使用定界符来包裹字符串

    <br>

    定界符内的所有內容都是字符串的內容，包括注释

    <br>

    定界符的起始符号后不能有任何信息，包括空格

    <br>

    定界符的结束符号必须顶格

    <br>

    定界符的结束符号后不能有任何信息	

    <br>

    heredoc 结构，等价于双引号，会将其中的变量解析

    <br>

    nowdoc 结构，等价与单引号

<br>

~~~php
// heredoc 结构

$str = <<< END
fun
END;

// nowdoc 结构

$str = <<< 'END'
fun
END;
~~~

<br>

## 数组型 ：array

<br>

​多个元素的序列组合

<br>

数组由元素组成

<br>

​数组的访问通过下标实现

<br>

​数组索引自增长是从当前元素所占的索引最大值开始

<br>

​PHP 数组的元素没有数量限制，也可以是任意数据类型，并且可以动态增加元素

<br>

​元素由下标以及值组成

<br>

索引数组 ：数组的下标都是数字

<br>

关联数组 ：数组的下标都是字符串

<br>

混合数组 ：数组的下标既有数字又有字符串

<br>

### 定义数组

<br>

定义空数组

<br>

~~~php
$arr = array();		// 第一种方式
$arr = [];		    // 第二种方式
~~~

<br>

定义非空数组

<br>

~~~php
$arr = [1];		    // 定义单个元素
$arr = [1, 2, 3];    // 定义多个元素
$arr = ["name" => "x", "age" => "18"];		// 指定下标定义数组，下标可以是数字也可以是字符串
~~~

<br>

### 添加元素

<br>

~~~php
$arr[] = "peter";	      // 系统自动生成下标
$arr["name"] = "peter";	  // 指定下标
~~~

<br>

### 访问元素

<br>

~~~php
echo $arr[0];
echo $arr['name'];
~~~

<br>

### 修改元素

<br>

~~~php
$arr[0] = 'mark';
~~~

<br>

### 删除数据

<br>

~~~php
unset($arr[0]);		// 删除一个元素
unset($arr);	    // 删除整个数组
~~~

<br>

### 二维数组

<br>

二维数组中的元素本身就是一个数组（ PHP 中理论上不限制数组的层级，当元素本事是一个二维数组时，就是多维数组 ）

<br>

可以有多维数组，但是层级尽量不要超过三（ 效率低 ）

## 数据类型转换

<br>

### 自动转换 

<br>

将值取出然后改变类型，不会影响原有的数据类型

<br>

### 强制转换 

<br>

资源和 NULL 不能强制转换

<br>

不改变原来数据

<br>

```php
(boolea)$a		// 将变量a的值取出然后转为布尔值
```

<br>

改变原来数据

<br>

~~~php
styletype($a, 'bool')	// 将变量a的结果转为布尔值
~~~
